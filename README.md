# mpfk.github.io — deprecated

This repository previously hosted **www.mattlydon.me** via GitHub Pages.

The site has been migrated to **AWS (S3 + CloudFront, deployed with AWS CDK)** and is now
maintained in **[Mpfk/www.mattlydon.me](https://github.com/Mpfk/www.mattlydon.me)**.

GitHub does not permit disabling Pages for a user site (`<username>.github.io`), so this
repo now serves only a redirect to <https://www.mattlydon.me/>. Do not add content here —
make changes in the new repository instead.
